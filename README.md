## Description

Basic currency conversion service

## Usage

GET http://localhost:3000/quote?base_currency=USD&quote_currency=ILS&base_amount=1500

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
